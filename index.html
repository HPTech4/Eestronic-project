Artificial Intelligence

AI Based Cold Storage System

This project features a deep learning model that recognizes the type of perishable fruit stored and automatically adjusts the temperature (using a Peltier cooler) to an optimal range for extended freshness. It operates on a 12V 72,000mAh battery, making it highly energy-efficient. The system is integrated with a mobile app for remote monitoring of temperature, humidity, and fruit ripeness.

AI Based Shear Nut Picker Robot

A robotic system embedded with an arm and a deep learning object detection model that identifies shear nuts and picks them into a collection bin autonomously.

AI Groundnut Farm Weeding Drone

A drone equipped with an AI groundnut leaf detector to distinguish groundnut plants from weeds. The drone automatically sprays herbicide when a weed is detected and stops when a groundnut leaf is identified.

AI Based Yam and Maize Disease Detection and Recommendation Device

Device that incorporate Linear SVM model deployed on ESP32 which predict the type of Disease possibly to affect a Yam or Maize in a storage room base on the temperature and humidity value of the storage room.

AI Based Fertilizer Advisor

A software and a device designed to Advise the farm based on the NPK values,temperature,humidity and Moisture content of the soil.The software can be used on desktop while the device can be installed on a farm for a real time solution.
